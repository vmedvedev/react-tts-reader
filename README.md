## Getting started
A simple client Text To Speech Reader.

### Build app and run it locally
***

#### Build docker image:
```bash
$ docker-compose build
```

#### Run server:
```bash
$ docker-compose up
```

#### Play with it in the web browser:
[http://localhost:3000](http://localhost:3000)
